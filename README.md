# 🚀 Premium Frontend Landing Page

A modern, responsive, animation-rich landing page built using HTML, CSS, JavaScript, and GSAP.

This project focuses on creating a premium user experience through smooth animations, responsive layouts, interactive sections, and modern UI design principles.
